Página Web Personal 🌐

Diseño y publicación de mi primera página web personal como espacio para proyectos, innovación y desarrollo tecnológico.

Desarrollé una página web personal desde una idea inicial partiend de un boceto a mano hasta una versión publicada con GitHub Pages. El proceso combinó diseño visual, edición manual de código, apoyo de herramientas de inteligencia artificial y organización de subpáginas para construir un portafolio académico y tecnológico propio.

🎯 Objetivo

Crear una página web personal que funcione como portafolio digital, permitiendo mostrar proyectos, trayectoria académica, intereses técnicos y contenido propio de forma ordenada, visualmente coherente y accesible desde internet.

  • Presentar perfil académico y proyectos
  • Diseño visual coherente y propio
  • Navegación clara entre subpáginas
  • Publicación accesible en línea

📋 Descripción del proyecto ◆

Desarrollé mi primera página web personal con el objetivo de construir un espacio propio para presentar mi perfil académico, proyectos, intereses en física, robótica, innovación y desarrollo tecnológico.


El proceso comenzó con una etapa de diseño conceptual a mano, donde definí las secciones principales, colores, estilo visual y estructura general del sitio. Luego utilicé herramientas de inteligencia artificial como ChatGPT y Claude para transformar esas ideas iniciales en propuestas visuales, código HTML/CSS y mejoras progresivas en el diseño.


A partir de esas versiones fui editando manualmente la página, ajustando textos, navegación, estructura, colores y detalles visuales hasta obtener una versión funcional. Finalmente, publiqué el sitio usando GitHub Pages, dejando abierta la posibilidad de explorar en una etapa futura el despliegue desde una Raspberry Pi 5.
